### 3. Create and Deploy an Application on Google App Engine to Display Even Numbers

Algorithm:
1. Start.
2. Open Google Cloud Console and sign in.
3. Create a new project and enable App Engine.
4. Create application files:
   - main.py (Flask app to generate even numbers)
   - app.yaml (runtime configuration)
   - requirements.txt (Flask dependency)
5. In main.py, define a route (/) to accept input n and generate even numbers.
6. Configure App Engine using app.yaml.
7. Deploy the application using gcloud app deploy.
8. Access the deployed URL in a browser.
9. Pass n as a query parameter to view even numbers in JSON format.
10. End.

from flask import Flask, request, jsonify

app = Flask(__name__)

@app.route("/")
def generate_evens():
    try:
        # default is 10
        n = int(request.args.get("n", 10)) 
        
        if n < 1:
            return jsonify({"error": "n must be >= 1"}), 400
            
        # Generating even numbers
        even_numbers = [2*i for i in range(1, n + 1)]
        
        return jsonify({
            "count": n,
            "even_numbers": even_numbers
        })
        
    except ValueError:
        return jsonify({"error": "Invalid input. Provide an integer n."}), 400

if __name__ == "__main__":
    app.run(host="0.0.0.0", port=8080)
